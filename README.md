# VIPProgressView
某种进度状态显示控件

![](http://7xuwb4.com1.z0.glb.clouddn.com/QQ%E6%88%AA%E5%9B%BE20160530160310.png?imageView2/1/w/400/h/400)

----------

基本可以显示两种状态，各种尺寸图标以垂直居中状态显示。支持动态添加图标，进度。
控件会将设置的图标的前两项作为常用图标，后两项作为最终图标。

优势：

- 支持自定义图标显示
- 支持自定义进度颜色
- 支持自定义数据大小
- 所有元素以居中显示

不足：

- 不支持自定义文字
- 目前仅支持两类图标

使用者可以根据自身需求自行更改绘制测量逻辑。
